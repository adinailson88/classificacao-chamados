OBJETIVO
Criar um projeto novo, separado do Malha IA operacional, para executar experimento de classificacao e reclassificacao automatica de chamados conforme roteiro metodologico.

DECISOES CONFIRMADAS PELO USUARIO
1. Trabalhar no novo repositorio GitHub chamado classificacao-chamados, identificado no GitHub como adinailson88/classifica-o-chamados.
2. Usar pasta local na Area de Trabalho.
3. A nova planilha vai somente ate a coluna M; colunas apos M estao vazias.
4. O script deve considerar somente linhas nao vazias.
5. A quantidade de linhas pode aumentar; nao assumir total fixo.
6. A planilha sera tratada no Google Drive/Google Sheets.
7. Podem ser criadas abas na propria planilha quando necessario, mas ainda nao foi autorizado escrever nela.
8. O usuario ainda nao liberou app token/credencial para essa planilha.
9. A cada etapa importante, criar um arquivo TXT de continuidade na pasta da Area de Trabalho para permitir continuacao por outra IA.

ESTADO TECNICO DO SCRIPT ATUAL
O script src/validar_planilha_experimento.py:
1. le config_experimento.json;
2. aceita teste offline via --arquivo-valores;
3. aceita leitura Google Sheets via --credenciais;
4. valida cabecalho esperado;
5. conta somente linhas nao vazias;
6. usa range A:M;
7. nao escreve na planilha.

PROXIMO PASSO RECOMENDADO
1. Usuario liberar credencial/token Google Sheets para a planilha.
2. Executar:
   python src\validar_planilha_experimento.py --credenciais CAMINHO_DO_JSON
3. Confirmar cabecalho real e total de linhas nao vazias.
4. Depois disso, criar script seguro para criar abas experimentais, ainda sem executar classificacao.
